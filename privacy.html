AegisX Privacy Policy

Effective Date: February 2026

1. Introduction

AegisX ("the Bot") is a moderation, protection, and community management system for Discord servers. This Privacy Policy explains what data is collected and how it is used.

2. Information Collected

The Bot may collect and store the following data:

  • Discord User IDs
  • Discord Server IDs
  • Channel IDs
  • Role IDs
  • Message IDs (only when required for moderation, tickets, or giveaways)
  • Ticket and giveaway configuration data

The Bot does NOT collect:

  • Passwords
  • Email addresses
  • Payment information
  • Private messages outside of bot interactions

3. How Data Is Used

Collected data is used strictly for:

  • Moderation systems
  • Ticket system functionality
  • Giveaway system functionality
  • Command execution
  • Server configuration storage

4. Data Storage

Data is stored securely and is not sold, rented, or shared with third parties.

Only the developer has access to stored data for maintenance and support purposes.

5. Data Removal

Server owners may request removal of stored server data by contacting the developer.

6. Security

Reasonable measures are taken to protect stored data. However, no system can guarantee 100% security.

7. Changes to This Policy

This Privacy Policy may be updated at any time. Continued use of the Bot constitutes acceptance of any changes.
